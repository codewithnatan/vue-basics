# 🟢 Vue 3 Learning Basics

This repository contains my very first stepps, experiments, and fundamental concepts inside the **Vue.js 3** framework. It serves as my practical playground to master reactivity and framework mechanics.

## 🧠 What I Learned
* **Project Setup:** Initializing a clean, modern frontend workspace using. **Vite**.
* **Reactivity (`ref`):** Learning how Vue automatically syncs JavaScript variables with the HTML template [🔍].
* **The `.value` Mechanic:** Learning why Vue protects reactive variables inside an object wrapper and how to safely modify them in `<script setup>` [🔍].
* **Event Handling (`@click`):** Creating interactive buttons to change data states instantly.

# 🛠️ Created Components
* **Interactive Counter:** A dual-button component capable of incrementing and decremennting values in real-time while logging errors safely to the browser console.

# 💻 Tech Stack & Tools
* **Framework:** Vue 3 (Composition API / `<script setup>`)
* **Build Tool:** Vite
* **IDE Extension:** Vue - Official (Syntax Highliting & IntelliSense) [🔍]
